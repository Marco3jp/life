sample

sampleは例示するためのObjectである。このオブジェクトはTemplateの子として持たれることを前提として作られている。

    Sample

    • 構文: new Sample(option)
    • パラメータ:
      • {undefined | Option} option
    • 説明: 
      // デフォルト値で動かす場合
let sample = new Sample();
// カスタム値で動かす場合
const option = { userName: "foge man" };
let customSample = new Sample(option);

      Sampleクラスのコンストラクタ。オプションによって初期値やフラグをセットすることが出来る。オプションの詳細は該当ドキュメントを参照。

    setFlag

    • 構文: setFlag(flagName, value)
    • パラメータ:
      • string flagName
      • {undefined | boolean} value
    • 説明: 
      // ドキュメント化が終わったフラグを立てる
sample.setFlag("documented");
// 実は終わってなかった
sample.setFlag("documented", false);

      フラグを立てるメソッド。Valueで任意の値を設定できるが、removeFlag()を用いてFlagを折ることも出来る(何らかの値をそのままFlagに落としたいときにこちらのメソッドを用いると良い)。

    removeFlag

    • 構文: removeFlag(flagName)
    • パラメータ:
      • string flagName
    • 説明: 
      // 新たにドキュメント化する部分ができた
sample.removeFlag("documented");

      フラグを折るメソッド。
