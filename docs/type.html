型

型名は全て大文字始まりである。Classが重複していても型名は _をPrefixとしてしない場合がある(例: _locationとLocation)ので注意。

?で始まる型はundefinedable、|で併記されているものはどちらかを示す。

    Action

    • プロパティ
      • {number} id
      • {string} name
      • {Function} require
      • {string} requireText
      • {Function} effect
      • {string} effectText
      • {?Script} script

    Event

    • プロパティ
      • {number} id
      • {string} name
      • {Function} require
      • {string} requireText
      • {Function} effect
      • {string} effectText
      • {?number} weight
        • イベントの発生する重み
        • 複数のイベントが発生するとき、この値によってソートされる
        • デフォルト値は0
      • {?Script} script

    Item

    • プロパティ
      • {number} id
      • {string} name
      • {Function} require
      • {string} requireText
      • {Function} effect
      • {string} effectText
      • {?Script} script

    Script

    この型は特に仕様が確定していません。他のプロパティで参照されている箇所もありますが、変更の可能性があることを十分に気をつけてください。

    • プロパティ
      • {?object} texts
      • {?object} operations
      • {?ViewScript} viewScript
      • {?Function} createViewScript

    Location

    • プロパティ
      • {number} id
      • {string} name
      • {Function} require
      • {string} requireText
      • {coordinate} coordinate
      • {Area} area
      • {?Script} script
      • {?Array<Location>} childLocation

    Coordinate

    • プロパティ
      • {number} x
      • {number} y
      • {?number} z

    Area

    • プロパティ
      • {number} x
      • {number} y
      • {?(number | LocationHeight)} z

    LocationHeight

    • プロパティ
      • {number} positive
      • {?number} negative

    Option

    • プロパティ
      • {?string} version
      • {?string} resourceVersion
      • {?ResourceURIList} resourceList
      • {?Setting} gameSetting

    ResourceURIList

    • プロパティ
      • {?Array<string>} actionURIList
      • {?Array<string>} characterURIList
      • {?Array<string>} eventURIList
      • {?Array<string>} itemURIList
      • {?Array<string>} locationURIList

    HandlerDefineder

    • プロパティ
      • {string} key
      • {?Function} validator

    InventoryRecord

    • プロパティ

    この型は一切仕様が確定していません。他のプロパティで参照されている箇所もありますが、変更の可能性があることを十分に気をつけてください。

    • プロパティ
